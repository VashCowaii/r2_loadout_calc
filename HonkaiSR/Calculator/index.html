Rotations Overview
Rotations are, in part, already automatic. However, if you've played the game, you know that auto-play can be dumb as fuck and sometimes the same can apply here.

How do we fix that? User-defined conditions.

User-defined conditions allow you to set dynamic rotations for your characters.
- Want Robin to wait before using her team-wide advance?
- Want Archer to wait for more skill points?
- Want Saber to only Ult when Sunday's buffs are up?

These allow you to impose any restriction to achieve the aforementioned results. If a condition doesn't exist that you want, join the discord and let me know.

FILTERS & SETTINGS
TAGS
Tag Filters are used to retrieve any and all items that ever use the stats you specify here. These apply first when used. Enter manually, or use presets. Or both. Both is good.

Smart Filters do NOT work if you filter by tags.

EXCLUSIONS & INCLUSIONS
These filters are used to forcibly exclude or include specified items from generation.
Exclusions are used when you do not want to see an item(s) from a category in your results.
Inclusions are used when you ONLY want to see an item(s) from a category in your results.
